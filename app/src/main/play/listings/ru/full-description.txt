Основные возможности:
  • НЕ требует root-прав
  • Использует обычные файлы конфигурации tinc.
  • Настраиваемый доступ/обход VPN для каждого приложения Android.
  • Поддержка приватных ключей защищенных паролем.
  • Управление подключением через графический интерфейс.
  • Автоматизируется с помощью сторонних приложений Android через Intents.

Примечания:
  • Это приложение представляет собой всего лишь VPN-клиент, который по умолчанию не привязан ни к какому поставщику услуг VPN.
  • Для подключения требуется существующая сеть tinc.
  • Дополнительную информацию о протоколе tinc VPN см. на https://www.tinc-vpn.org/.

Ограничения клиента специфичные для платформы Android:
  • Работает с tinc только в режиме роутера.
  • Одновременно подключается только к одной сети tinc.
  • Нет поддержки скриптов-хуков.
